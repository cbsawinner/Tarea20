Acotaciones sobre el proyecto:
Se lanza la activity principal que tiene un botón que nos lleva a la activity clientes,
en ella podemos
Limpiar: las cajas de Texto.
Guardar: un Registro en la base de datos.
Actualizar: un registro existente.
Buscar: al escribir un código de cliente buscará los datos de este y llenará las cajas de texto con
la información.
Listado: este nos mostrará todos los datos ingresados.
Eliminar: un Registro existente (No se implementó la confirmación de eliminación)
Mapas: Según la dirección y la ciudad, convertirá estos datos en coordenadas las que luego serán
representadas a través de un marcador en el mapa usando la API de google maps.
Tamaño para Tablet de 7": se ha creado la carpeta layout-sw600dp,
que permite soporte para ese tipo de dispositivos, además de la reestructuración 
de botones con imágenes que permita un diseño de botones en una fila en vez de dos
como se muestra en el teléfono.


En la carpeta bin va el apk instable para tablet o teléfono.
Se cambia de fondo a la aplicación, se añade un icono de la aplicación también.

Se cambia la internacionalización a dos idiomas Español-Inglés
